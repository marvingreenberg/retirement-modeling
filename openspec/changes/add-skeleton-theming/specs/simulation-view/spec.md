## MODIFIED Requirements

### Requirement: Themed simulation controls and results
The Simulate view buttons, summary statistics panel, and year-by-year detail table SHALL use Skeleton/Tailwind styling consistent with the active theme and mode.

#### Scenario: Run button styled
- **WHEN** the Simulate view renders
- **THEN** the "Run Simulation" button uses Skeleton preset button classes

#### Scenario: Summary panel styled
- **WHEN** simulation results are displayed
- **THEN** summary statistics appear in a themed card/surface container

#### Scenario: Detail table styled
- **WHEN** the year-by-year table renders
- **THEN** the table uses Skeleton table classes with appropriate header and row styling
